# 🔐 Simple Password Generator

A lightweight and easy-to-use password generator that creates secure and customizable passwords instantly.

---

## 🚀 Features

- Generate strong random passwords
- Adjustable password length
- Include:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special characters (!@#$%^&*)
- One-click copy to clipboard
- Simple and clean UI
